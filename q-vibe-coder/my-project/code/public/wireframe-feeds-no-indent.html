Top Recommendations

Option 1 (Left Accent Bar) - Most common pattern. Used by Slack, Discord, VS Code. Clear grouping without indent.

Option 2 (Shared Background) - Very clear visual container. Both items obviously belong together.

Option 7 (Connected Backgrounds) - Elegant gradient effect that flows from parent to child.

Key point: All 10 options maintain the same left alignment. The hierarchy is communicated through color, backgrounds, borders, or typography - not indentation.
